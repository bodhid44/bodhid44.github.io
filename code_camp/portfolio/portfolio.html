Portfolio

CodeBreaker
PHP Project - Online Game

Web-based code-breaking game: written in PHP/Javascript. Compter generates random pick of four coloured elements from six possible colours, with duplicates allowed. Player than tries to guess the correct positions and colours. After each guess, the user selection is scored and the number of correct positions/colours is reported. Helper option available which highlights any previous guesses that conflict with the current guess.

Based on the original Mastermind game.
Study Log
Node.js/Express/MongoDB - Study Log Application

Web-based study log application: written in Node.js/Express/MongoDB (currently under development). Allows user to keep track of current study progress. Also enables user to store links to any useful study materials available online. User can also create own courses/areas of study and write, and display, markdown formatted notes for each area of study and/or study material.
